Wodel-Test

A Solution for Language Independent Mutation Testing

View on GitHub

This is an example of a Wodel program to replace a constructor call by a null literal:


generate 2 mutants
in "data/out/"
from "data/model/"
metamodel "data/model/java.ecore"
description "Other mutations"

with blocks {
	cir "Replaces a call to a constructor by null" {
		a = select one Assignment
			where {rightHandSide is typed ClassInstanceCreation}
		create NullLiteral in a->rightHandSide
	}
}

Transforms the return value of a function


rtstmt = select one ReturnStatement
  where {expression is typed NumberLiteral}
nl = select one NumberLiteral in rtstmt->expression
modify nl with {tokenValue = random-int-string(0, 9)}

Deletes a call to a void method


remove one MethodInvocation
  where {method->returnType->type is typed PrimitiveTypeVoid}

Deletes a call to a non-void method


remove one MethodInvocation
  where {method->returnType->type not typed PrimitiveTypeVoid}

Replaces a call to a constructor method by null


a =  select one Assignment where {rightHandSide is typed ClassInstanceCreation}
create NullLiteral in a->rightHandSide

Replace an assignment to a variable by the Java default values


remove one Assignment
   where {rightHandSide is typed NumberLiteral}
