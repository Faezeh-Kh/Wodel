Wodel applied to UML Class Diagrams. [UML Class Diagrams Ecore]

Adds a redundant constraint to the CD [uco1]:


    clone contents one Constraint

Adds an extraneous constraint to the CD [uco2]:


    create Constraint with {type = one Class, id = random-string(1, 4)}

Adds a redundant association to the CD [uas1]:


    clone one Association

Adds a redundant derived association to the CD [uas2]:


    a1 = select one Association
    a2 = select one Association where {src = a1->dest}
    a3 = create Association with {src = a1->src, dest = a2->dest}
    modify a3 with {copy(nsrc, a1.nsrc), copy(ndest, a2.ndest)}

Adds an extraneous association to the CD [uas3]:


    create Association with {src = one Class, dest = one Class, name = random-string(1, 4)}

Adds an extraneous derived association to the CD [uas4]:


    create Association with {src = one Class, dest = one Class, name = random-string(1, 4), derived = true}

Adds a redundant generalization to the CD [uge1]:


    cl = select one Class where {super <> null and super->super <> null}
    sup1 = select one Class in cl->super where {super <> null}
    sup2 = select one Class in sup1->super where {self <> cl->super}
    modify cl with {super = sup2}

Adds an extraneous generalization to the CD [uge2]:


    cl1 = select one Class where {super <> null}
    sup = select all Class in closure(cl1->super)
    cl2 = select one Class where {self <> sup}
    modify cl1 with {super = cl2}

Changes the constraint by deleting the references to a class Attribute [wco1]:


    remove reference attributes in one Constraint where {attributes <> null}

Changes the Attribute data type in the constraint [wco2]:


    c = select one Constraint where {attributes <> null}
    modify c->attributes with {type = other Class}

Change the constraint by deleting the calls to specific operation [wco3]:


	c = select one Constraint where {operations <> null}
	remove reference operations in c

Changes an arithmetic operator for another and supports binary operators: "add", "subtract", "multiply", "divide" [wco4]:


	c = select one Constraint where {operations <> null and operations->operators <> null}
	oper = select one Operation in c->operations where {operators <> null}
	op = select one Operator in oper->operators
	modify op with {operator in {add, subtract, multiply, divide}}

Changes the constraint by adding the conditional operator "not" [wco5]:


	c = select one Constraint where {operations <> null}
	oper = select one Operation in c->operations
	op = create Operator in oper->operators
	modify op with {operator in {not}}

Changes a conditional operator "and" for a conditional operator "or" [wco6a]:


	c = select one Constraint where {operations <> null and operations->operators <> null}
	oper = select one Operation in c->operations where {operators <> null}
	op = select one Operator in oper->operators where {operator = {^and}}
    modify op with {operator = {^or}}

Changes a conditional operator "or" for a conditional operator "and" [wco6b]:


	c = select one Constraint where {operations <> null and operations->operators <> null}
	oper = select one Operation in c->operations where {operators <> null}
	op = select one Operator in oper->operators where {operator = {^or}}
    modify op with {operator = {^and}}

Changes the constraint by deleting the conditional operator "not" [wco7]:


	c = select one Constraint where {operations <> null and operations->operators <> null}
	oper = select one Operation in c->operations where {operators <> null}
	op = select one Operator in oper->operators where {operator = {^not}}
	remove op

Changes a relational operator for another operators: "lt" , "lte", "gt" , "gte", "equals", "distinct" [wco8]:


	c = select one Constraint where {operations <> null and operations->operators <> null}
	oper = select one Operation in c->operations where {operators <> null}
	op = select one Operator in oper->operators where {operator in {lt, lte, gt, gte, ^equals, ^distinct}}
	modify op with {operator in {lt, lte, gt, gte, ^equals, ^distinct}}

Changes a constraint by deleting a unary arithmetic operator ("negative") [wco9]:


	c = select one Constraint where {operations <> null and operations->operators <> null}
	oper = select one Operation in c->operations where {operators <> null}
	op = select one Operator in oper->operators where {operator = {negative}}
	remove op
